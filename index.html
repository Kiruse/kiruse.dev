Penumbra DEX WIP

Penumbra is a DeFi protocol, a collection of Solidity smart contracts resembling an Automated Market Maker running on the Polygon Network.

Penumbra merges single-asset liquidity pools into the abstract Hyperpool: rather than pairs of specific fungible tokens, every type of asset can be paired with another, effectively maximizing liquidity depth across all assets.

Simultaneously, liquidity providers can stake a single token, reducing hassle and improving capital efficiency. LPs enjoy a portion of trading fees levied on any trade on the platform involving their staked assets.

The effects of the Hyperpool on returns are yet to be observed.

PyParFact WIP

ParFact is a Python library exposing an API to build untokenized and contextual formal language text parsers. ParFact parsers skip tokenization and build Abstract Syntax Trees directly.

ParFact is still an early Work in Progress. The inspiration for the project sprouted from a client project which saw me mine non-uniform textual data from Wiktionary. The WikiCode parser library I employed for the task was too generic and even a little buggy. I'd been meaning to develop such a parser generator before already, so I took this opportunity to start a side-side project.

Once complete, ParFact will excel at creating more natural programming languages, or parsing non-uniform data sources with plentiful special cases, such as WikiCode found on Wiktionary.

ParFact achieves its functionality through Deterministic Conversion - the process of converting a Non-Deterministic Finite Automaton into a determinstic one. A custom grammar notation allows it to understand interdependencies between rules.

Voxel Populi Engine WIP

A very early Work in Progress, Voxel Populi Engine is an Open Source collection of Julia libraries composing my custom game engine developed from scratch, built atop OpenGL 3.3+, GLFW, and TrueType.

As it is currently a long-term hobby project, unfortunately, the engine is not advanced enough for any visual demonstration. I am also in the process of a massive refactoring, which means parts of the existing codebase are currently private.

With this game engine, I wish to explore applications of machine learning in game development & design. Specifically, I see massive potential for procedural content generation, adaptive difficulty, XR, and animation tuning - though there are endlessly more possibilities.

Pia.jl frozen

Pia is the peer-to-peer networking component of VPE above. It is designed to allow developers to establish an "internet of peers" - whether it is for gaming, distributed computing, or decentralized networks. As a crypto enthusiast, it has built-in support for crypto-identities and cryptocurrencies - albeit fully optional.

As VPE is far from its need for netcode still, Pia is currently merely a proof of concept, albeit an incomplete one.
